svn cleanup 报错

发表于 2019-10-31 | 更新于: 2019-10-31 | 分类于 软件使用
字数统计: 106 | 阅读时长 ≈ 1 | 阅读:
  1. 更新的文件被打开导致失败，然后运行“cleanup”时出现了
    Previous operation has not finished; run 'cleanup' if it was interrupted
  2. 在svn工程目录下，有一个隐藏目录”.svn”，在这个目录下有一个wc.db文件。
  3. 用sqlite3.exe 打开 wc.db文件
  4. 查看记录
    select * from work_queue;
  5. 把列表清空
    delete from work_queue;
  6. 再次查询，确认是否清空
    select * from work_queue;
  7. 然后，再 clean up清空
