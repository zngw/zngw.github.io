使用sfntly提取ttf字体

发表于 2019-11-06 | 更新于: 2019-11-06 | 分类于 软件使用
字数统计: 7k | 阅读时长 ≈ 23 | 阅读:

  • 在游戏中经常需要用到各种字体，但是一些字体都是比较大的。我们只需要提取自己需要的一些字符就行了。

  • sfntly是一款谷歌开源的字体提取工具
    下载地址：https://github.com/googlefonts/sfntly

  • 编译
    用Intellij IDEA打开java工程目录，设置Artifacts为SfntTool，直接编译

  • 找到Artifacts输出目录中的sfntly.jar

  • 查看说明
    java -jar sfntly.jar -h

    F:\sfntly-master\java\classes\artifacts\sfntly_jar>java -jar sfntly.jar -h
Subset [-?|-h|-help] [-b] [-s string] fontfile outfile
Prototype font subsetter
      -?,-help        print this help information
      -s,-string       String to subset
      -b,-bench        Benchmark (run 10000 iterations)
      -h,-hints        Strip hints
      -w,-woff         Output WOFF format
      -e,-eot  Output EOT format
      -x,-mtx  Enable Microtype Express compression for EOT format
F:\sfntly-master\java\classes\artifacts\sfntly_jar>

  • 提取字体

java -jar sfnttool.jar -s '需要提取的字体' 源字体库 导出的最终字体库

F:\sfntly-master\java\classes\artifacts\sfntly_jar>java -jar sfntly.jar  -s '这是一段测试文字' Alibaba-PuHuiTi-Medium.ttf Ali.ttf
